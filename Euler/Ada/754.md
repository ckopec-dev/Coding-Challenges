# Euler Problem 754 Solution in Ada

I'll solve Euler Problem 754, which asks for the number of ways to tile a 3×n rectangle with 2×1 dominoes.

Let me break this down step by step:

## Problem Understanding

We need to find the number of ways to tile a 3×n rectangle using 2×1 dominoes. Each domino covers exactly 2 unit squares.

## Mathematical Approach

This is a classic dynamic programming problem. Let's define:
- f(n) = number of ways to tile a 3×n rectangle
- We can use the recurrence relation approach

Let's think about how we can fill the first column:
1. Place 3 vertical dominoes (covers 3×1 area)
2. Place 2 horizontal dominoes in the first 2 rows (covers 2×2 area)
3. Place 2 horizontal dominoes in the last 2 rows (covers 2×2 area)
4. Place 1 horizontal domino in the middle row and 2 vertical dominoes (covers 3×2 area)

After working through the recurrence relations, the pattern emerges as:
f(n) = f(n-1) + 5×f(n-2) + f(n-3) - f(n-4) for n ≥ 4

With base cases:
- f(0) = 1 (empty rectangle)
- f(1) = 1 (3×1 rectangle can only be filled with 3 vertical dominoes)
- f(2) = 5 (2×3 rectangle has 5 different tilings)
- f(3) = 11 (3×3 rectangle has 11 different tilings)

procedure Euler_754 is
   type Long_Integer is range -2**63 .. 2**63-1;
   
   function Tiling_Count(n : Positive) return Long_Integer is
      f : array(0 .. n) of Long_Integer;
   begin
      if n = 0 then
         return 1;
      elsif n = 1 then
         return 1;
      elsif n = 2 then
         return 5;
      elsif n = 3 then
         return 11;
      end if;
      
      f(0) := 1;
      f(1) := 1;
      f(2) := 5;
      f(3) := 11;
      
      for i in 4 .. n loop
         f(i) := f(i-1) + 5 * f(i-2) + f(i-3) - f(i-4);
      end loop;
      
      return f(n);
   end Tiling_Count;
   
   Result : Long_Integer;
