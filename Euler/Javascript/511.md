    let result = n;
    for (let i = 2; i * i <= n; i++) {
        if (n % i === 0) {

                n /= i;
            }
            result -= result / i;
        }
    }

    }

}

## Mathematical Analysis

Euler Problem 511 asks us to find numbers n where φ(n) = φ(n+1) = φ(n+2), where φ is Euler's totient function.

Key insights:
1. Euler's totient function φ(n) counts the number of integers ≤ n that are coprime to n
2. This condition is extremely rare - it occurs for very specific patterns of numbers
3. The solution involves finding numbers where three consecutive integers all have the same totient value
