# Euler Problem 807 Solution in Scala

I'll solve Euler Problem 807, which asks for the sum of all distinct values of a^b + b^a where a and b are positive integers with 1 ≤ a, b ≤ 100.

```scala
object Euler807 {
  def main(args: Array[String]): Unit = {
    val limit = 100
    val distinctValues = scala.collection.mutable.Set[Long]()
    
    // Calculate a^b + b^a for all combinations
    for (a <- 1 to limit; b <- 1 to limit) {
      val result = BigInt(a).pow(b) + BigInt(b).pow(a)
      distinctValues += result
    }
    
    val sum = distinctValues.sum
    println(s"Sum of all distinct values: $sum")
  }
}
```

However, since the numbers involved can be very large, let me provide a more robust solution that handles the large integers properly:

```scala
object Euler807 {
  def main(args: Array[String]): Unit = {
    val limit = 100
    val distinctValues = scala.collection.mutable.Set[BigInt]()
    
    // Calculate a^b + b^a for all combinations
    for (a <- 1 to limit; b <- 1 to limit) {
      val aToB = BigInt(a).pow(b)
      val bToA = BigInt(b).pow(a)
      val result = aToB + bToA
      distinctValues += result
    }
    
    val sum = distinctValues.sum
    println(s"Sum of all distinct values: $sum")
  }
}
```

And here's an even more concise functional approach:

```scala
object Euler807 {
  def main(args: Array[String]): Unit = {
    val limit = 100
    
    val sum = (1 to limit).flatMap { a =>
      (1 to limit).map { b =>
        BigInt(a).pow(b) + BigInt(b).pow(a)
      }
    }.toSet.sum
    
    println(s"Sum of all distinct values: $sum")
  }
}
```

## Explanation:

1. **Problem Understanding**: We need to find all distinct values of the form a^b + b^a where both a and b range from 1 to 100.

2. **Approach**: 
   - Generate all combinations of (a,b) where 1 ≤ a,b ≤ 100
   - For each pair, calculate a^b + b^a
   - Keep only distinct values
   - Sum all distinct values

3. **Key Considerations**:
   - The numbers can get very large (like 100^100), so we use `BigInt`
   - We use a Set to automatically handle duplicates
   - The solution uses Scala's functional programming features for clean code

4. **Time Complexity**: O(n²) where n=100, which is acceptable for this problem.

The final answer will be the sum of all distinct values of a^b + b^a for 1 ≤ a, b ≤ 100.
