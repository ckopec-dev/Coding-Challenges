# Euler Problem 877 Solution in Lightning Web Component

I'll solve Euler Problem 877 using Lightning Web Components. This problem involves finding the sum of all primes that are substrings of a given number.

## Problem Statement
Find the sum of all primes that are substrings of the number 12345678901234567890.

## LWC Implementation

```javascript
// euler877.js
import { LightningElement } from 'lwc';

export default class Euler877 extends LightningElement {
    result = 0;
    primes = [];
    number = '12345678901234567890';

    connectedCallback() {
        this.solve();
    }

    solve() {
        // Generate all substrings
        const substrings = this.getAllSubstrings(this.number);
        
        // Find all primes among substrings
        const primeSubstrings = [];
        
        for (let substring of substrings) {
            // Skip leading zeros
            if (substring.length > 1 && substring[0] === '0') {
                continue;
            }
            
            const num = parseInt(substring);
            if (this.isPrime(num)) {
                primeSubstrings.push(num);
            }
        }
        
        // Remove duplicates and sum
        const uniquePrimes = [...new Set(primeSubstrings)];
        this.primes = uniquePrimes.sort((a, b) => a - b);
        this.result = uniquePrimes.reduce((sum, prime) => sum + prime, 0);
    }

    getAllSubstrings(str) {
        const substrings = [];
        for (let i = 0; i < str.length; i++) {
            for (let j = i + 1; j <= str.length; j++) {
                substrings.push(str.substring(i, j));
            }
        }
        return substrings;
    }

    isPrime(num) {
        if (num < 2) return false;
        if (num === 2) return true;
        if (num % 2 === 0) return false;
        
        const sqrt = Math.sqrt(num);
        for (let i = 3; i <= sqrt; i += 2) {
            if (num % i === 0) return false;
        }
        return true;
    }

    get formattedResult() {
        return this.result.toLocaleString();
    }

    get primeCount() {
        return this.primes.length;
    }
}
```
