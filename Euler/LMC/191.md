# Euler Problem 191 Solution in Little Man Computer

Euler Problem 191 asks: "Based on the rules for a school attendance record, how many valid records of length 30 exist?"

Let me break down the problem and solve it in LMC:

## Problem Analysis
We need to count valid attendance records of length 30 where:
- 'A' = Absent (max 1 allowed)
- 'L' = Late (max 2 consecutive allowed)
- 'P' = Present (no restrictions)

## Solution Approach
I'll use dynamic programming with states representing:
- Number of absences (0 or 1)
- Number of consecutive lates (0, 1, or 2)
- Position in the sequence

## Explanation of the Algorithm

The solution uses a 2D DP approach with states:
- `DP00`, `DP01`, `DP02`: Number of valid sequences with 0 absences and 0, 1, or 2 consecutive lates
- `DP10`, `DP11`, `DP12`: Number of valid sequences with 1 absence and 0, 1, or 2 consecutive lates

For each position:
1. We can add a 'P' (present) - this resets consecutive lates to 0
2. We can add an 'L' (late) - this increments consecutive lates (if < 2)
3. We can add an 'A' (absent) - this can only be done once

The final answer is stored in the `RESULT` variable.
