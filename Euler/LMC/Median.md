# Euler Problem: Median - Little Man Computer Solution

## Problem Statement
Find the median of the numbers 1 to 1000, where the median is the middle value when the numbers are arranged in order.

## Solution Approach
In Little Man Computer, I'll:
1. Store numbers 1-1000 in memory
2. Sort them using bubble sort
3. Find the middle element (500th element for 1000 numbers)
